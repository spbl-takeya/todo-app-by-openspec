# Change: Todoアプリの基本機能を追加

## Why
Todoアプリケーションの初期実装。ユーザーが基本的なCRUD操作でタスクを管理できるようにする。

## What Changes
- タイトルと説明（任意）でTodo項目を作成できる機能を追加
- Todo項目を一覧表示できる機能を追加
- Todo項目の完了/未完了を切り替えられる機能を追加
- 既存のTodo項目を編集できる機能を追加
- Todo項目を削除できる機能を追加
- 完了状態でフィルタリングできる機能を追加

## Impact
- Affected specs: `todo-management`（新規capability）
- Affected code: 新規アプリケーション
